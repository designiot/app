#Design IoT APP

##Setup

1.Install Deps

    npm install cordova -g
    npm install ionic -g
    npm install
    bower install

2.Debug in PC
    
    ionic serve
    
    
3.Run
    
    ionic platform add android
    ionic run android
    
    
##Plugin Lists

 - cordova-plugin-bluetooth-serial
